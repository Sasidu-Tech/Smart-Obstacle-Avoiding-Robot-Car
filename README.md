# 🤖 Smart Obstacle Avoiding Robot Car

An autonomous obstacle-avoiding robot built using **Arduino Uno**, **L293D Motor Driver**, **HC-SR04 Ultrasonic Sensor**, and a **Servo Motor**. The robot continuously scans its surroundings, detects obstacles, and automatically selects the safest direction to move without human control. 0

---

# 🚀 Features

- Autonomous obstacle detection
- Intelligent left/right path selection
- Real-time distance measurement
- Servo-based ultrasonic scanning
- Four-wheel drive system
- Smooth motor control
- Rechargeable battery powered
- Easy to modify Arduino code

# ⚙️ Working Principle

1. Robot moves forward.
2. Ultrasonic sensor continuously measures distance.
3. When an obstacle is detected, the robot stops.
4. Servo rotates the ultrasonic sensor to scan both left and right.
5. Arduino compares both distances.
6. Robot turns toward the side with more free space.
7. Movement continues automatically. 1
